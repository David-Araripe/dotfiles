powerline needs to be installed so that it works with vim. Here's how:
# On Ubuntu/Debian:
sudo apt install powerline vim-powerline fonts-powerline

# On Fedora/RHEL/CentOS:
sudo dnf install powerline vim-powerline powerline-fonts

# On Arch:
sudo pacman -S powerline powerline-fonts

# On macOS with Homebrew:
brew install powerline-go
# or
pip3 install powerline-status
